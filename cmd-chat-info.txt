## COMMAND CHAT

Daftar command:
- !asal
- !lahir
- !umur
- !pacar
- !web
- !hobi

Selain itu, kamu juga bisa menggunakan teks langsung tanpa chat unttuk bisa mendapatkan jawaban (contoh: "kamu sekarang umur berapa?" atau "halo")

Catatan penting:
- Tidak semua teks langsung dapat memberikan jawaban. Pastikan kata yang kamu gunakan benar!
- Data chat kamu tidak akan pernah disimpan didalam server apapun! Jadi, saat kamu refresh browser, data akan hiilang.
